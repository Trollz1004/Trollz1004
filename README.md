# Trollz1004 Repository

This repository contains various projects, including the Date App DAO Dashboard - a comprehensive, production-ready application with advanced features.

## Projects

### Date App DAO Dashboard

A production-ready, full-stack dashboard for a hypothetical "Date App DAO". It includes a backend API and a frontend application, with comprehensive features:

#### Key Features
- **Authentication:** Firebase auth with Two-Factor Authentication (2FA)
- **E-commerce:** Full shop with Square payment integration
- **Fundraising:** Campaign creation and management
- **Real-time Chat:** WebSocket-based messaging
- **Advanced Search:** Powerful search and filtering across all content
- **Analytics Dashboard:** Visual charts and business metrics
- **Activity Feed:** Timeline of user and system activities
- **Theme Toggle:** Dark/light mode with persistence
- **Admin Tools:** Comprehensive administrative controls

#### Technology Stack
- Frontend: React, TypeScript, Material-UI, Vite, Zustand
- Backend: Node.js, Express, TypeScript
- Database: Google Cloud Firestore
- Authentication: Firebase with 2FA (TOTP)
- Real-time: Socket.IO
- Charts: Recharts
- Payments: Square

For more information, see the [Date App DAO Dashboard README](date-app-dashboard/README.md).
