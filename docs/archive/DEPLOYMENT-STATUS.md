## ğŸ› TROUBLESHOOTING

### Deployment Fails

```bash
# Check logs
gcloud run services logs read teamclaude-backend

# Common issues:
# 1. Billing not enabled -> Enable in Cloud Console
# 2. APIs not enabled -> Run: gcloud services enable ...
# 3. Quota exceeded -> Request quota increase
```

### Can't Access URLs

```bash
# Check service status
gcloud run services list

# Check if service is public

```

### Payments Not Working

```bash

# Check Square credentials

  -H "Authorization: Bearer $SQUARE_ACCESS_TOKEN"
```
